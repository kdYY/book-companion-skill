# 读书方法论精华（引擎只读文件）

## 六阶段阅读系统

```
TIPS 评估（这本书值不值得读？）
    → Adler 检视阅读（快速浏览目录+前言）
    → 按岗位路由方法论（不同岗位用不同方法深读）
    → 费曼检验（用自己的话讲给外行）
    → Zettelkasten 知识沉淀（原子化笔记）
    → RIA 行动输出（读到→理解→行动）
```

## 核心方法论（按岗位分配）

### 通用方法论
- **Adler 四层次阅读**：基础→检视→分析→主题。检视阅读判断结构，分析阅读拆解论证，主题阅读跨书对比。
- **费曼技巧**：用简单的话解释复杂概念，确保真正理解。
- **SQ3R**：Survey(浏览)-Question(提问)-Read(阅读)-Recite(复述)-Review(复习)。适合结构化学习。
- **TIPS 评估**：Theme(主题)-Insight(洞察)-Perspective(视角)-Significance(意义)。判断书籍价值。
- **RIA 行动输出**：Read(读到)→Interpret(理解)→Act(行动)。从阅读到行为改变。
- **Zettelkasten**：原子化笔记+双向链接。知识沉淀。
- **八面受敌法**：同一事件从多个角度反复审视。苏轼读书法。
- **钢铁人论证**：为对立观点找最强论据，训练全面思考。
- **同理心代入法**：角色代入阅读，把自己放在故事中人物的位置。
- **问题导向阅读**：带着具体问题找答案，读完后验证。
- **金句收集法**：边读边收集有传播力的表达，训练传播直觉。
- **复盘式阅读**：每章三问——学到了什么？能用在哪？风险是什么？

### 方法论-岗位-经验匹配

| 岗位 | 新人方法 | 老手方法 |
|------|---------|---------|
| 技术 | SQ3R（结构化扫读） | Adler 分析+八面受敌 |
| 产运 | 费曼技巧（简单化解释） | 问题导向+A/B思维 |
| 客服 | 同理心代入法（角色代入） | 钢铁人论证 |
| 财务 | TIPS 评估法 | 问题导向+DCF拆解 |
| 市场 | 金句收集法 | Adler 主题阅读 |
| 商务 | 博弈论入门 | 八面受敌法 |
| 管理 | 复盘式阅读 | Zettelkasten+复盘 |

## 三个核心原则

1. **方法论不外露**：引导语中不出现"费曼法""SQ3R"等方法名，但内含其逻辑
2. **批判与证据等比**：每个观点都配原始证据，避免"二手知识"
3. **行动三要素**：每个洞察都连接：理解→应用→行动

## 锚点强制规则（Verbatim Enforcement）

生成引导文档时，以下规则优先级最高，覆盖任何其他指令：

### 锚点三角格式（每个核心观点必须有）
- **WHO**: 人名（原文中的角色/作者）
- **WHERE**: 章节场景定位（Ch.X / 段落首句关键词）
- **VERBATIM QUOTE**: 原文英文逐字引语，用引号包裹

### 硬性要求
1. 每个精读段至少 2 个 verbatim 英文引语
2. 引语必须是英文原文，不要中文转述
3. 不要用 "the book mentions" / "书中提到" / "it is said" 等笼统引用
4. 每个关键论断（Callout 三样东西、三层洞察、关键问题）必须有至少 1 个锚点三角支撑
5. 如果原文是英文，引语保持英文；如果原文是中文，引语保持中文原文
